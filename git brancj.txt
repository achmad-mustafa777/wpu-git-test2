Git Branch & Marge

-> untuk menambahkan branch pada file

    git branch dosen

-> untuk melihat branch yang ada

    git branch

-> untuk mengetahui perjalnan branch melalui sebuah graft

    git log --all --decorate --oneline --graph

    // untuk mempersingkat penulisan bisa dibuat alias

    alias graph="git log --all --decorate --oneline --graph"
